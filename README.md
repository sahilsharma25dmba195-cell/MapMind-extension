# 🧠 MapMind Extension — Turn Any Article into Structured Knowledge

MapMind is a browser extension that transforms long-form content into **organized, geography-first notes**—instantly.

Instead of passively reading articles, MapMind helps you **extract, structure, and store knowledge** in a way that actually sticks.

---

## 🚀 What It Does

With one click, MapMind:

* 📄 Extracts content from any article or webpage
* 🧩 Breaks it into **structured, meaningful notes**
* 🌍 Categorizes information by **region, country, and topic**
* 🏷️ Auto-generates relevant tags
* 📥 Lets you import everything directly into your MapMind workspace

---

Most tools help you **save content**.
MapMind helps you **understand and organize it.**

Perfect for:

* 📊 Business & finance learners

* 🧠 Curious minds who want structured knowledge
* 📚 Students preparing for competitive exams
* 📰 Newsletter readers (Morning Brew, Daily Upside, etc.)

---

## 🛠️ How It Works

1. Open any article (newsletters, blogs, etc.)
2. Click the **MapMind Extension**
3. Hit **“Extract & Import”**
4. Review structured notes
5. Import them into your map

That’s it — your knowledge is now **organized, searchable, and connected**.

## 🧑‍💻 Installation

1. Download the code as Zip or Clone this repository

3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select this folder
